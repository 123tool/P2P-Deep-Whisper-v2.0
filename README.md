# 🌑 DEEP-WHISPER v2.0 (Deep-Web Edition) 🌑
> **Next-Gen Anonymous P2P Local Chat with E2EE & Self-Destruct Logic.**

**DEEP-WHISPER v2.0** adalah evolusi dari sistem komunikasi lokal yang dirancang khusus untuk privasi ekstrem. Menggunakan estetika *Deep-Web Noir*, versi ini memperkenalkan enkripsi *End-to-End* (E2EE) tingkat militer yang memastikan pesan tidak dapat disadap bahkan oleh admin jaringan sekalipun.

---

## 💀 Fitur Utama (V2 Upgrade)
* **🔐 E2EE AES-256 Encryption:** Semua pesan dan file dienkripsi di browser menggunakan *Secret Key*. Server hanya berfungsi sebagai kurir data acak.
* **⏳ Self-Destruct Mechanism:** Pesan akan memudar dan hancur secara otomatis dari UI dalam 30 detik untuk menghindari *shoulder surfing*.
* **🚨 Panic Button:** Tombol darurat (atau tekan `ESC`) untuk menghapus sesi dan mengalihkan layar ke halaman aman secara instan.
* **👻 Ghost Identity:** Identitas unik anonim otomatis (`GHOST-XXXX`) yang berganti di setiap sesi baru.
* **📂 Encrypted File Sharing:** Kirim gambar dan dokumen secara aman; data diubah menjadi *encrypted blob* sebelum dikirim.
* **🌑 Deep-Web UI:** Antarmuka premium berbasis *JetBrains Mono* dengan aksen merah darah untuk visibilitas rendah dan kenyamanan maksimal.

---

## 🛠️ Persyaratan Sistem
* **Node.js** v16.x atau lebih baru.
* **NPM** v8.x atau lebih baru.
* **Modern Browser** (Chrome/Brave/Firefox) untuk dukungan Crypto API.
